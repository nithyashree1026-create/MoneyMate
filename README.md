💸 MoneyMate – Premium Money Management Web App

MoneyMate is a premium, aesthetic personal finance tracker designed to help users manage their daily income, expenses, savings goals, and budgets in a smart and visually appealing way.
This project is built using HTML, CSS, and JavaScript and works completely in the browser using LocalStorage – no backend required.

✨ Features
🧾 Income & Expense Tracking
Add income and expense transactions
Categorize transactions (Food, Travel, Shopping, etc.)
Date-wise tracking
Delete transactions anytime

Smart Dashboard
Total Balance
Total Income
Total Expenses
Budget Remaining
